# 运维与排障

## 常见问题

### 页面为空

- 检查知识源路径是否正确
- 检查是否读取到了 Markdown 文件
- 检查 parser 是否因 frontmatter 异常全部降级

### 搜索结果不对

- 检查快照是否刷新
- 检查摘要和 tags 是否成功提取
- 第二阶段检查数据库全文索引

### 图谱异常

- 检查内部链接解析规则
- 检查 rawTarget 是否被错误过滤

## 重建策略

- 第一阶段：重启服务或清空缓存
- 第二阶段：触发某个 source 的全量 sync

## 日志建议

- source 级别日志
- 文件级别日志
- parser 异常日志
- 同步任务状态日志

## 备份建议

- 数据库每日备份
- source 配置单独备份
- 同步日志保留最近 30 天
